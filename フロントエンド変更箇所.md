## 2. app.js の変更

`fetch` を `authFetch` に置き換えるだけ。
これで 401 が返ってきたら自動でログインページへ遷移する。

以下の4箇所を検索して置き換え:

| 変更前 | 変更後 |
|--------|--------|
| `const response = await fetch('/api/library')` | `const response = await authFetch('/api/library')` |
| `const response = await fetch('/api/library/refresh')` | `const response = await authFetch('/api/library/refresh')` |
| `const response = await fetch(\`/api/volumes?` | `const response = await authFetch(\`/api/volumes?` |
| `const response = await fetch(\`/api/images?` | `const response = await authFetch(\`/api/images?` |
| `const response = await fetch('/api/library/status')` | `const response = await authFetch('/api/library/status')` |

### VS Code で一括置換する方法

1. `Ctrl+Shift+H` で検索・置換パネルを開く
2. 検索: `await fetch('/api/`
3. 置換: `await authFetch('/api/`
4. 「すべて置換」をクリック
